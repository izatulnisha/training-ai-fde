This repository, **training-ai-fde**, is organized as a 12‑week course for AI FDE training. Each week’s materials live in their own folder (`week-00` … `week-12`), while the main `README.md` serves as an index linking to them. The structure is meant to provide a clear progression from setup and foundations through advanced topics and a capstone project, making it easy to navigate and build upon as the course unfolds.
